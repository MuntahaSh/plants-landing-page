A modern, fully responsive **one-page website** for showcasing plants and gardening products. Built with **TailwindCSS 4** and plain JavaScript, this landing page features a clean design, smooth interactions, and easy-to-update content.

## Features

- Fully responsive design (mobile & desktop)
- Clean, modern layout with TailwindCSS
- Home, About, Popular Plants, Customer Reviews, and Footer sections
- Embedded images and icons
- SwiperJS slider for customer reviews
- Smooth scroll and animations using ScrollReveal
- Easy-to-update HTML, CSS, and JS files

  ## Technologies Used

- HTML5
- TailwindCSS 4
- Vanilla JavaScript
- SwiperJS
- Remix Icon CDN
